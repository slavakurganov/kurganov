# ReportsCreator

## Описание
Консольная утилита для создания структуры папок Reports/ГГГГ/ММ и сохранения отчета.

## Запуск
dotnet run

## Пример вывода
=== СОЗДАНИЕ СТРУКТУРЫ ПАПОК ===

Будет создана папка: Reports\2026\06

[OK] Папка создана
[OK] Отчет сохранен: Reports\2026\06\report_20260619_021921.txt

=== ДЕРЕВО ПАПОК ===

Reports/
└── 2026/
    └── 06/
        ├── report_20260619_021042.txt
        ├── report_20260619_021456.txt
        └── report_20260619_021921.txt

## Структура
- Program.cs - основной код
- Reports/ - создаваемая структура
